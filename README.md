# Parallel Wave Function Collapse

## Setup
> **Pre-requisite**: pre-commit

To get started, you can run:
```sh
cargo install --path .
pre-commit install
```
or simply run the setup script
```sh
chmod +x setup.sh && ./setup.sh
```
